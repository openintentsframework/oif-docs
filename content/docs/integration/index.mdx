# Integration Guide

Choose your integration path based on what you're building.

## Integration Paths

### For Wallets & DApps
Integrate OIF APIs to offer cross-chain swaps to your users.  
**Start here**: [Integration Guide](/docs/integration/overview)

### For Protocols
Deploy your own aggregator instance for complete control.  
**Start here**: [Aggregator Documentation](/docs/aggregators)

### For Solvers
Earn fees by fulfilling user intents across chains.  
**Start here**: [Solver Documentation](/docs/solvers)
